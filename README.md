# KUC-MADDPG
Source code for the paper entitled "KUC-MADDPG: A KL-Unlearning and Partial Communication Based Framework for Task Offloading in Vehicular Edge Computing" submitted to IEEE for possible publication.

# Requirements
•	Python 3.10.19

•	Pytorch 2.0.1

•	Numpy 1.24.4

# Functions
•	partial_commu: The partial communication mechanisms in algorithms.

•	train_MADDPG.py: The deep reinforcement learning algorithm based on MADDPG used in this experiment.

•	KL_Unlearning.py: KL-Unlearning of the bad strategies learned by the MADDPG algorithm

•	test_KUC_MADDPG.py: The performance and comparison of KUC-MADDPG algorithm.

# Usage
Directly run the train~. py file, then run the tset~. py file
